# GitHub 文案包

这份文件用于固定“推演局”在 GitHub 上的推荐文案，方便后续统一 README、仓库简介和对外介绍口径。

## 仓库名称

`tuiyanju`

## 仓库简介（About）

多 Agent 认知推演系统：输入一个议题，自动完成冲突拆解、动态选角、暗房独白、圆桌激辩与结构化真相报告导出。

## 一句话版本

推演局不是普通问答页，而是一台把“一个问题”拆成“多方处境图”的推演终端。

## 稍长版本

推演局（Dynamic Cognitive Orchestrator）是一个多 Agent 认知推演系统。用户输入一个议题后，系统会自动拆解冲突轴、生成不同利益结构的虚拟角色，让他们在互相隔离的暗房中独立发言，并最终由局长整合为一份结构化、可阅读、可传播、可导出的正式报告。
