## 稍长版本

推演局（Dynamic Cognitive Orchestrator）是一个多 Agent 认知推演系统。用户输入一个议题后，系统会自动拆解冲突轴、生成不同利益结构的虚拟角色，让他们在互相隔离的暗房中独立发言，并最终由局长整合为一份结构化、可阅读、可传播、可导出的正式报告。

### 版本 A：作品发布型

刚把一个我很喜欢的项目整理到了 GitHub：

**推演局 / Dynamic Cognitive Orchestrator**

它不是普通 AI 问答页。  
你输入一个议题，系统会先拆冲突轴，再动态选角，让多个互相隔离的 Agent 进入“暗房”独立发言；在 `Pro / Ultra` 档位下，还会进入圆桌激辩，最后由“局长”输出一份结构化真相报告。

我想做的不是“更会回答问题的 AI”，而是一个能把复杂议题拆成立体处境图的推演终端。
