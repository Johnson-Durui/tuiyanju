# 推演局 · 导出说明

## 导出模式

当前 PDF 导出提供两种模式：

### `full`
- 面向完整阅读、留档和深度复盘
- 默认结构：
  - 封面
  - 30 秒结论页
  - 目录与模块总览
  - 推演过程提示
  - 选角总览
  - 独立思考暗房
  - 圆桌激辩（如有）
  - 局长整合报告

### `share`
- 面向转发传播、社交分享和快速阅读
- 默认结构：
  - 封面
  - 30 秒结论页
  - 选角总览（压缩版）
  - 暗房金句精选
  - 圆桌激辩精选片段
  - 局长报告核心结论
  - 3 条行动建议

## 设计原则

- `full` 保留完整推演链路，适合认真通读和归档
- `share` 不直接塞入整段长独白，而是从暗房、圆桌和局长报告中抽取最有代表性的句子和结论
- 两种模式保持统一视觉风格，但 `share` 会明显更紧凑，适合快速转发

## 前端按钮

- `导出完整 PDF`：下载 `full` 模式报告
- `导出传播版 PDF`：下载 `share` 模式报告

## 当前实现说明

- `share` 模式不依赖后端新增字段
- 所有传播版内容均从现有正式输出中提炼，不额外生成另一套逻辑
- 传播版文件名格式为：`推演局-{话题名}-share.pdf`
